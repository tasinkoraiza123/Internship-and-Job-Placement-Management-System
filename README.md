Internship & Job Placement Management System

Description: 
A web-based system to manage student internships, job applications, and placement records.

Features:
- Student & company registration
- Internship/job posting
- Application tracking
- Eligibility based on CGPA and department

Technologies Used:
- PHP
- MySQL
- HTML, CSS, JavaScript
- XAMPP

How to Run:
1. Install XAMPP
2. Start Apache & MySQL
3. Import the database
4. Run the project on localhost
